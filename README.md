# Java 8 Features - Examples

A collection of Java 8 feature examples followed along with a Udemy course.

## Topics Covered

- **Lambdas** - Lambda expressions and syntax
- **Functional Interfaces** - Consumer, Predicate, Function, Supplier
- **Method References** - Static, instance, and constructor references
- **Streams** - Stream API and operations
- **Terminal Operations** - collect, reduce, count, findAny, etc.
- **Numeric Streams** - IntStream, LongStream, DoubleStream
- **Parallel Streams** - Parallel processing with streams
- **Optional** - Avoiding null pointer exceptions
- **Date & Time API** - LocalDate, LocalTime, LocalDateTime
- **Default Methods** - Default methods in interfaces
- **Imperative vs Declarative** - Comparison of programming styles

## Prerequisites

- Java 8 or higher
- Any Java IDE (IntelliJ IDEA / Eclipse)

1. Clone the repository
   ```bash

   ```
2. Open in your IDE
3. Run any class with a `main` method
